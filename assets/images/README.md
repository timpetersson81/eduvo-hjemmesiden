# Logo Placeholder

Dette er en placeholder for logoet. 

For at tilføje dit logo:
1. Gem dit logo som `logo.png` i `assets/images/` mappen
2. Logoet vil automatisk vises i header og footer
3. Logoet er responsivt og tilpasser sig til forskellige skærmstørrelser

Logo indstillinger:
- Header størrelse: 40px høj (32px på mobil)
- Footer størrelse: 32px høj
- Format: PNG anbefalet for transparent baggrund
- Anbefalet maks bredde: 200px for bedste resultat
